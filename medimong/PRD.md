# 임상실습 AI 부조 시스템

## 문제
> 의대 본과 3학년 임상실습생이 매일 반복하는 자료 준비·기록 작성·교수 커뮤니케이션을 AI 스킬 + 오케스트레이션으로 자동화하여, 학습에 집중할 시간을 확보한다

- **현재 상태**: 실습 건당 자료 준비·정리에 1시간 이상 소요. 선행/복습 자료 찾기, 의무기록지 작성은 수작업. 교수님 이메일 확인·답장, 실습 일정 관리도 각각 따로 처리
- **원하는 상태**: 1주 후 — 하나의 명령으로 "내일 실습 준비 → 선행 자료 생성 → 일정 캘린더 등록 → 교수님 메일 확인·답장"이 한 번에 돌아가는 상태
- **성공 기준**: 건당 처리 시간 1시간 → 15분 이내, 주 5회 이상 실제 사용

## 스킬
| # | 스킬명 | 한 줄 설명 | 상태 |
|---|--------|-----------|------|
| 1 | `/my-context-sync` | 내일 시간표 기반 선행 자료 + 오늘 복습 자료 자동 생성 | ✅ 동작 |
| 2 | `/student-medical-record` | 환자 문진 PDF/텍스트 → 내과 학생의무기록지 docx 생성 | ✅ 동작 |
| 3 | `/clerkship-orchestrator` | Gmail 확인 + 캘린더 일정 등록 + 교수님 답장 + 선행자료를 하나로 묶는 오케스트레이션 | 🔨 진행중 |

## 변화 기록
- **Day 1**: "임상실습 자료 정리를 자동화하는 웹앱을 만들자"
- **Day 6**: "웹앱 대신 Claude Code 스킬로 전환. 개별 스킬은 동작하지만, Gmail·캘린더·답장까지 묶는 오케스트레이션은 아직 미구현"
- **가장 크게 달라진 점**: 무거운 웹앱 → 가벼운 스킬 조합으로 전환. 그리고 단순 자료 생성을 넘어 "교수님 커뮤니케이션 + 일정 관리"까지 포함하는 통합 자동화로 범위가 확장됨
